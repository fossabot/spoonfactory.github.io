JavaScript / Full-stack Developer

I’m multi talented developer who likes to keep eye on the quality of the products we develop. That’s how I become our Lead QA at Solinor. Still I’m one of the developers but on my responsible is to keep our testing methodologies up-to-date, Jenkins CI servers properly configured and all around quality on top notch level.

I have developed with bunch of languages – in JavaScript, PHP, Python, SQL, Perl, Shell Scripts and C#. My favourite one is JavaScript and I been using it almost in every project I been part of. Still I have been unfortunate because projects I have been usually use something, which is not using most modern technics at the certain moment. For example I have used Sencha Ext JS framework.

I have become familiar with jQuery and vanilla JavaScript over the years. Node.js haven’t been my tool at work but have been practising it at my spare time.

I want to educate my self all the time and been learning Perl at evenings. It’s not most developers’ favourite thing but I want to learn different kind of thinking. Perl is also powerful tool to help with Shell Scripting. Besides I want to learn Clojure but it might be my summer holiday project…

Past few years I have been full-stack developer with DevOps skills. It’s my passion to develop amazing things. I’m not bound to any specific programming language or framework. I want to learn new and become better with skills I already have.
